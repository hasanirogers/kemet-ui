# Kemet Carousel

The Carousel allows you to click or swipe through content.


## How to use

```html
<kemet-carousel>
  <kemet-carousel-slide slot="slides">Slide 1</kemet-carousel-slide>
  <kemet-carousel-slide slot="slides">Slide 2</kemet-carousel-slide>
  <kemet-carousel-slide slot="slides">Slide 3</kemet-carousel-slide>
</kemet-carousel>
```

## API

### &lt;kemet-carousel&gt;
The main carousel element.
<ArgsTable of="kemet-carousel" />

#### Options

The options object contains a number of settings for the carousel. Use these options to set the default options for the carousel. You can overrride them at specified breakpoints in the breakpoints object.

| option    | description                                                                                            |
|-----------|------------------------------------------------------------------------------------------------------- |
| perView   | The number of slides to display in the container. Access decimals numbers.                             |
| perMove   | The number of slides to move on next and prev shifts.                                                  |
| gap       | The number, in pixels, of space between each slide.                                                    |
| slideshow | The number, in seconds, to display a slide before rotating to a new one. Use 0 to disable a slideshow. |
| rewind    | Loops back to the first or last slide when a limit is reached.                                         |
| center    | Centers the active slide.                                                                              |

#### Slots

| slot        | description                                                                               |
|-------------|------------------------------------------------------------------------------------------ |
| slides      | Place the slide elements here.                                                            |
| toolbar     | Allows you to construct a toolbar within the carousel. Place things like pagination here. |
| information | Allows you to enter custom HTML such as a slide description to be used in the toolbar.    |

#### CSS

| css                                             | description                               | default                            |
| ----------------------------------------------- | ----------------------------------------- | ---------------------------------- |
| --kemet-carousel-width                          | The width of the carousel.                | 100%                               |
| --kemet-carousel-height                         | The height of the carousel.               | auto                               |
| --kemet-carousel-toolbar-gap                    | The space between items in the toolbar.   | 1rem                               |
| --kemet-carousel-toolbar-justify-content        | The cross axis alignment of the toolbar.  | center                             |
| --kemet-carousel-toolbar-padding                | The padding of the toolbar.               | 1rem                               |
| --kemet-carousel-toolbar-inner-width            | The inner toolbar width.                  | 100%                               |
| --kemet-carousel-toolbar-inner-bottom           | The bottom position of the inner toolbar. | 0                                  |
| --kemet-carousel-toolbar-inner-top              | The top position of the inner toolbar.    | auto                               |
| --kemet-carousel-toolbar-inner-color            | The inner toolbar text color.             | var(--kemet-color-white)           |
| --kemet-carousel-toolbar-inner-background-color | The inner toolbar background color.       | rgba(0, 0, 0, 0.4)                 |
| --kemet-carousel-slides-border                  | The border around the slides.             | 1px solid var(--kemet-color-gray1) |
| --kemet-carousel-transition-speed               | The transition speed of the slides.       | 300ms                              |
| --kemet-carousel-arrows-opacity                 | The arrows start opacity.                 | 0.25                               |
| --kemet-carousel-arrows-opacity-hover           | The arrows however opacity.               | 1                                  |
| --kemet-carousel-arrows-transition-speed        | The arrows transition speed.              | 300ms                              |
| ::part(container)                               | Contains the slides and toolbar.          |                                    |
| ::part(slides)                                  | Wrapper for the slider.                   |                                    |
| ::part(slider)                                  | Contains the slides.                      |                                    |
| ::part(toolbar)                                 | Contains the toolbar.                     |                                    |
