The Carousel allows you to click or swipe through content.

    ### &lt;kemet-carousel&gt;
    The main carousel element.
    <ArgTypes of="kemet-carousel" />

    #### Options

    The options object contains a number of settings for the carousel. Use these options to set the default options for the carousel. You can overrride them at specified breakpoints in the breakpoints object.

    <table>
      <thead>
        <tr>
          <th>Option</th>
          <th>Description</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>perView</td>
          <td>The number of slides to display in the container. Access decimals numbers.</td>
        </tr>
        <tr>
          <td>perMove</td>
          <td>The number of slides to move on next and prev shifts.</td>
        </tr>
        <tr>
          <td>gap</td>
          <td>The number, in pixels, of space between each slide.</td>
        </tr>
        <tr>
          <td>slideshow</td>
          <td>The number, in seconds, to display a slide before rotating to a new one. Use 0 to disable a slideshow.</td>
        </tr>
        <tr>
          <td>rewind</td>
          <td>Loops back to the first or last slide when a limit is reached.</td>
        </tr>
        <tr>
          <td>center</td>
          <td>Centers the active slide.</td>
        </tr>
      </tbody>
    </table>

    ### &lt;kemet-carousel-link&gt;
    Zero-based index link to a slide in the carousel.
    <ArgTypes of="kemet-carousel-link" />


    ### &lt;kemet-carouse-next&gt;
    A link to the next slide in the carousel.
    <ArgTypes of="kemet-carousel-next" />

    ### &lt;kemet-carousel-prev&gt;
    A link to the previous slide in the carousel.
    <ArgTypes of="kemet-carousel-prev" />

    ### &lt;kemet-carousel-slide&gt;
    A container for a slide in the carousel.
    <ArgTypes of="kemet-carousel-slide" />

    ### &lt;kemet-carousel-first&gt;
    A link to the first slide in the carousel.
    <ArgTypes of="kemet-carousel-first" />

    ### &lt;kemet-carousel-last&gt;
    A link to the last slide in the carousel.
    <ArgTypes of="kemet-carousel-last" />

    ### &lt;kemet-carousel-current&gt;
    Displays the active slide number.
    <ArgTypes of="kemet-carousel-current" />

    ### &lt;kemet-carousel-total&gt;
    Displays the total slide number.
    <ArgTypes of="kemet-carousel-total" />

    Note that this technically isn't a web component. Use this custom tag name to grab information form a slide and display it in the toolbar.
