# Kemet Button

A versatile button that can be used either to submit a form, trigger an action, or link to content.


## How to use

```html
<kemet-button>Button</kemet-button>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-button.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-button.js" type="module"></script>
```

### Slots

| slot  | description                                                 |
|-------|------------------------------------------------------------ |
| left  | Alows you to place an icon to the left of the button text.  |
| right | Alows you to place an icon to the right of the button text. |
