# Kemet Sortable

A list that can be sorted by drag and drop.


## How to use

```html
<kemet-sortable>
  <kemet-sortable-item>Item 1</kemet-sortable-item>
  <kemet-sortable-item>Item 2</kemet-sortable-item>
  <kemet-sortable-item>Item 3</kemet-sortable-item>
</kemet-sortable>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-sortable/kemet-sortable.js';
import 'kemet-ui/dist/components/kemet-sortable/kemet-sortable-item.js';
```
