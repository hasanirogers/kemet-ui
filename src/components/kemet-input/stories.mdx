# Kemet Input

An enhanced input element.


## How to use

```html
<kemet-input></kemet-input>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-input/kemet-input.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-input/kemet-input.js" type="module"></script>
```

#### Methods

| method           | description                             |

| css                                    | description                         | default                              |

| --kemet-input-border                   | The border of the input.            | 1px solid var(--kemet-color-primary) |
| --kemet-input-border-color-error       | The border of the error state.      | 1px solid var(--kemet-color-error)   |
| --kemet-input-border-color-success     | The border of the success state.    | 1px solid var(--kemet-color-success) |
