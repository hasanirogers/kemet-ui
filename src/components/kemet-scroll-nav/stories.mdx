# Scroll Nav

An element that transforms at a scroll point.

## How to use

```html
<kemet-scroll-nav>
  [Your nav markup here]
</kemet-scroll-nav>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-scroll-nav/kemet-scroll-nav.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-scroll-nav/kemet-scroll-nav.js" type="module"></script>
```
