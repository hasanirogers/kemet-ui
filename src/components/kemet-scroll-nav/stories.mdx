# Kemet Drawer

An element that transforms at a scroll point.

## How to use

```html
<kemet-scroll-nav>
  [Your nav markup here]
</kemet-scroll-nav>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-scroll-nav.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-scroll-nav.js" type="module"></script>
```

### CSS

| css                                          | description                                 | default   |
| -------------------------------------------- | ------------------------------------------- | --------- |
| --kemet-scroll-nav-padding                   | The padding of the nav.                     | 1rem 2rem |
| --kemet-scroll-nav-background                | The background color of the nav.            | #fafafa   |
| --kemet-scroll-nav-transition                | The transition speed of the transformation. | 300ms     |
| --kemet-scroll-nav-resize-height             | The height of the pre-transformed nav.      | 400px     |
| --kemet-scroll-nav-resize-height-transformed | The height of the transformed nav.          | 100px     |
