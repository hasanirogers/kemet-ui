# Kemet Count

Maintains a character count for a input field. Is to be used only in the component slot of a Field component.

<kemet-tabs panel-effect="stacked" divider>

    ### Markup
    ```html
    <kemet-field slug="unique-identifier">
      <kemet-input slot="input" name="input-field"></kemet-input>
      <kemet-count slot="component" message="characters remaining." limit="32"></kemet-count>
    </kemet-field>
    ```

    ```javascript
    import 'kemet-ui/dist/components/kemet-count/kemet-count.js';
    ```
