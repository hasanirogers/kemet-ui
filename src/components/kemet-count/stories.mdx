# Kemet Count

Maintains a character count for a input field. Is to be used only in the commponent slot of a Field component.


## How to use

```html
<kemet-field slug="unique-identifier"">
  <kemet-input slot="input" name="input-field"></kemet-input>
  <kemet-count slot="component" message="characters remaining." limit="32"></kemet-count>
</kemet-field>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-count/kemet-count.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-count/kemet-count.js" type="module"></script>
```
