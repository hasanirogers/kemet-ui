# Kemet Tabs

A collection of components to enable tab behavior.

## How to use

```html
<kemet-tabs>
  <kemet-tab slot="tab">One</kemet-tab>
  <kemet-tab slot="tab">Two</kemet-tab>
  <kemet-tab slot="tab">Three</kemet-tab>
  <kemet-tab-panel slot="panel">Panel One</kemet-tab-panel>
  <kemet-tab-panel slot="panel">Panel Two</kemet-tab-panel>
  <kemet-tab-panel slot="panel">Panel Three</kemet-tab-panel>
</kemet-tabs>
```

## How to import

import 'kemet-ui/dist/components/kemet-tabs/kemet-tabs.js';

## API

### &lt;kemet-tabs&gt;

<ArgsTable of="kemet-tabs" />

#### Slots

| slot  | description            |
| ----- | ---------------------- |
| tab   | Place the tabs here.   |
| panel | Place the panels here. |

#### CSS

| css                           | description                         | default                    |
| ----------------------------- | ----------------------------------- | -------------------------- |
| --kemet-tabs-ink-size         | The thickness of the ink.           | 6px                        |
| --kemet-tabs-ink-radius       | The radius on the ink.              | 6px                        |
| --kemet-tabs-ink-color        | The color of the ink.               | var(--kemet-color-primary) |
| --kemet-tabs-divider-size     | The thickness of the divider.       | 1px                        |
| --kemet-tabs-divider-color    | The color of the divider.           | var(--kemet-color-gray4)   |
| --kemet-tabs-transition-speed | The transition speed of the panels. | 0.5s                       |
| --kemet-tabs-spacer           | The space between tabs and panels.  | 1rem                       |
| ::part(links)                 | The container for the tablist.      |                            |
| ::part(tablist)               | Contains the tabs.                  |                            |
| ::part(panels)                | Contains the panels.                |                            |
| ::part(ink)                   | The ink element.                    |                            |
| ::part(divider)               | The divider element.                |                            |
