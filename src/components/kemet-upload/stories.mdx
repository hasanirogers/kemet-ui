# Kemet Upload

An interface for uploading files (actual upload script must be handled by your app).


## How to use

```html
<kemet-upload>
  <kemet-upload-file></kemet-upload-file>
  <kemet-upload-file></kemet-upload-file>
  <kemet-upload-file></kemet-upload-file>
</kemet-upload>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-upload/kemet-upload.js';
import 'kemet-ui/dist/components/kemet-upload/kemet-upload-file.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-upload/kemet-upload.js" type="module"></script>
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-upload/kemet-upload-file.js" type="module"></script>
```

## Stories

**Note** Kemet UI cannot provide actual upload functionality for you. You need to write your own upload script. You can do this by listening to the `kemet-upload-change` event. This event will give you the files that have been added. You'll need to dynamically insert a `kemet-upload-file` element for each of these files in the event. An example upload script is coming shortly.
