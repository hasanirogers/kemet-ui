# Kemet Icon

An element that represents an icon from a predefined set of open sourced icons.


## How to use

```html
<kemet-icon set="[open source icon set]" icon="[icon name]" size="[icon size]"></kemet-icon>
```

## Sets

Kemet Icon comes with several predefined sets of icons that can be referenced. These icons come from open source projects. To get more information on the icons please visit the set's documentation site.
