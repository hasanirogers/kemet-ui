# Kemet FAB

The FAB, or Floating Action Button, performs a primary action on a page.


## How to use

```html
<kemet-fab>
  <kemet-icon slot="icon" icon="[chose-an-icon]"></kemet-icon>
  Label Here
</kemet-fab>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-fab/kemet-fab.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-fab/kemet-fab.js" type="module"></script>
```

#### CSS

| css                          | description                        | default                              |
| ---------------------------- | ---------------------------------- | ------------------------------------ |
| --kemet-fab-size             | The width and height of the fab.   | 50px                                 |
| --kemet-fab-color            | The text color of the fab.         | var(--kemet-color-white)             |
| --kemet-fab-background-color | The bg color of the tab.           | var(--kemet-color-primary)           |
| --kemet-fab-outlined-color   | The text color of an outlined fab. | var(--kemet-color-primary)           |
| --kemet-fab-outlined-border  | The border of an outlined fab.     | 1px solid var(--kemet-color-primary) |
| --kemet-fab-pill-radius      | The border radius of a pill fab.   | 10rem                                |
