# Kemet FAB

The FAB, or Floating Action Button, performs a primary action on a page.


## How to use

```html
<kemet-fab>
  <kemet-icon slot="icon" icon="[chose-an-icon]"></kemet-icon>
  Label Here
</kemet-fab>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-fab/kemet-fab.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-fab/kemet-fab.js" type="module"></script>
```
