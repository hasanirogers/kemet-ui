# Kemet Card

A highly configurable panel design to display media and information.


## How to use

```html
<kemet-card>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</kemet-card>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-card/kemet-card.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-card/kemet-card.js" type="module"></script>
```
