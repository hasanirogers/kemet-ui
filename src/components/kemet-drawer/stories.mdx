# Kemet Drawer

A component that adds an off-canvas menu with different effects.

## How to use

```html
<kemet-drawer>
  <nav slot="navigation">Your navigation goes here.</nav>
  <section slot="content">Your content goes here.</section>
</kemet-drawer>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-drawer/kemet-drawer.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-drawer/kemet-drawer.js" type="module"></script>
```
