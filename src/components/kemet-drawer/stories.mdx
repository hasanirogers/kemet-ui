# Kemet Drawer

A component that adds an off-canvas menu with different effects.

## How to use

```html
<kemet-drawer>
  <nav slot="navigation">Your navigation goes here.</nav>
  <section slot="content">Your content goes here.</section>
</kemet-drawer>
```
