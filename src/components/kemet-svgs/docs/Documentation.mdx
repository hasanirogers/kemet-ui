# Kemet SVGs

A set of SVG graphics that can be referenced any where in a page or app.

<kemet-tabs divider panel-effect="stacked">

  <kemet-tab-panel slot="panel">
    ### Markup
    ```html
    <!-- define the set of svgs here -->
    <kemet-svgs>
      <svg>
        <defs>
          <g id="[name of svg]">
            [svg paths here]
          </g>
        </defs>
      </svg>
    </kemet-svgs>
    ...
    <!-- reference a svg from the set here -->
    <kemet-svg svg="[name of svg]" viewbox="[svg source viewbox value]"></kemet-svg>
    ```

    **Note:** if your svg does not have a viewbox value set use the width and height of the svg as the last two numbers for the view box. An svg with a width of 1920 and height of 1080 for example should have a viewbox value of `0 0 1920 1080`.


    ### Import JavaScript
    ```javascript
    import 'kemet-ui/dist/components/kemet-svgs/kemet-svgs.js';
    import 'kemet-ui/dist/components/kemet-svgs/kemet-svg.js';
    ```

    ### Import HTML
    ```html
    <script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-svgs/kemet-svgs.js" type="module"></script>
    <script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-svgs/kemet-svg.js" type="module"></script>
    ```
