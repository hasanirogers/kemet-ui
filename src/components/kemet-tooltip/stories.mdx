# Kemet Tooltip

The Tooltip is a Popper with an additional arrow pointing to the trigger.


## How to use

```html
<kemet-tooltip>
  <kemet-button slot="trigger">[trigger]</kemet-button>
  <div slot="content">[content]</div>
</kemet-tooltip>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-tooltip/kemet-tooltip.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-tooltip/kemet-tooltip.js" type="module"></script>
```
