# Kemet Field

Used in combination with Input, Select, and Textarea, to make a Field.


## How to use

```html
<kemet-field label="The Label">
  <kemet-input slot="input" name="input"></kemet-input>
</kemet-field>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-field/kemet-field.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-field/kemet-field.js" type="module"></script>
```
