# KemetPopover

Displays an element in position of a trigger. Also acts as a tooltip.


## How to use

```html
<kemet-popover>
  <button slot="trigger">Popover Trigger</button>
  <span slot="content">Your popover contents here. Supports HTML.</span>
</kemet-popover>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-popover/kemet-popover.js';
import 'kemet-ui/dist/components/kemet-popover/kemet-popover-close.js';

```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-popover/kemet-popover.js" type="module"></script>
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-popover/kemet-popover-close.js" type="module"></script>
```
