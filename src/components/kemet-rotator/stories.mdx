# Kemet Rotator

A component that rotates through an array of text.

## How to use

```html
<kemet-rotator messages='["my", "text", "here"]'></kemet-rotator>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-rotator/kemet-rotator.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-rotator/kemet-rotator.js" type="module"></script>
```
