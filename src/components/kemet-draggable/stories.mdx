# Kemet Draggable

A draggable element with the ability to remember its position.

## How to use

```html
<kemet-draggable memory="a-unique-identifier">
  <kemet-button>Drag Me!</kemet-button>
</kemet-draggable>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-draggable/kemet-draggable.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-draggable/kemet-draggable.js" type="module"></script>
```
