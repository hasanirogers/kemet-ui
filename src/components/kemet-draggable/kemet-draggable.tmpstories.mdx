# KemetDraggable

A draggable element with the ability to remember its position.


## How to use

### Installation

```bash
npm install @kemet/kemet-draggable
```

```js
import '@kemet/kemet-draggable/kemet-draggable.js';
```

### Basic Usage
```html
<kemet-draggable measure memory="a-unique-identifier">
  <button>Drag Me!</button>
</kemet-draggable>
```
