# Kemet Radio

An enhanced radio button.


## How to use

```html
<kemet-radios legend="The legend.">
  <kemet-radio value="1" name="kemet-radio-button" label="Item 1"></kemet-radio>
  <kemet-radio value="2" name="kemet-radio-button" label="Item 2"></kemet-radio>
  <kemet-radio value="3" name="kemet-radio-button" label="Item 3"></kemet-radio>
</kemet-radios>
```

## How to import

<br />

```javascript
import 'kemet-ui/dist/components/kemet-radio/kemet-radios.js';
```

### &lt;kemet-radio&gt;

<ArgsTable of="kemet-radio" />

#### Methods

| method     | description                              |
| ---------- | ---------------------------------------- |
| .blur()    | Simulates a blur on the input element.   |
| .click()   | Simulates a click on the input element.  |
| .focus()   | Simulates a click on the input element.  |

#### CSS

| css                            | description                                 | default                              |
| ------------------------------ | ------------------------------------------- | ------------------------------------ |
| --kemet-radio-size             | The size of the radio button.               | 18px                                 |
| --kemet-radio-border           | The outer border of the radio button.       | 1px solid var(--kemet-color-primary) |
| --kemet-radio-dot-color        | The color of the radio button's dot.        | var(--kemet-color-primary)           |
| --kemet-radio-dot-border-width | The border width of the radio button's dot. | 3px                                  |
| --kemet-radio-dot-border-color | The border color of the radio button's dot. | var(--kemet-color-white)             |
| --kemet-radio-dot-color-filled | The filled color of the radio button.       | var(--kemet-color-white)             |
| ::part(label)                  | The label that contains the radio button.   |                                      |
| ::part(button)                 | The radio button.                           |                                      |
| ::part(text)                   | The text next to the radio button.          |                                      |
| ::part(dot)                    | The circle that fills the radio button.     |                                      |
