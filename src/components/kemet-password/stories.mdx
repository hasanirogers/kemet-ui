# Kemet Combo

Gauges the strength of a password entered by the user.


## How to use

```html
<kemet-field>
  <kemet-input slot="input" type="password" name="input"></kemet-input>
  <kemet-password slot="component"></kemet-password>
</kemet-field>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-password/kemet-password.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-password/kemet-password.js" type="module"></script>
```
