# KemetScrollLink

A component that adds a smooth scrolling link to an target.


## How to use

### Installation

```bash
npm install @kemet/kemet-scroll-link
```

```js
import '@kemet/kemet-scroll-link/kemet-scroll-link.js';
```

### Basic Usage
```html
<kemet-scroll-link>
  <button>Scroll to Top</button>
</kemet-scroll-link>
```
