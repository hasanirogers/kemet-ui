# Kemet Scroll Link

A component that adds a smooth scrolling link to an target.


## How to use

### HTML

```html
<article>
  <h1>Heading</h1>
  ...
<article>
<kemet-scroll-link>
  <button>Scroll to Top</button>
</kemet-scroll-link>
```

### JavaScript
```js
const link = document.querySelector('kemet-scroll-link');
link.target = document.querySelector('h1');
```

Use the target property on `kemet-scroll-link` to point to a DOM element that should be the destination of the link. In this example, we link to the first `h1`.
