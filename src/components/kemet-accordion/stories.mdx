# Kemet Accordion

A component that acts like a standard accordion.


## How to use

```html
<kemet-accordion>
  <kemet-accordion-panel>
    <h3 slot="trigger">Trigger 1</h3>
    <kemet-icon slot="icon" icon="caret-down" size="18"></kemet-icon>
    <div slot="body">The quick brown fox jumps over the lazy dog.</div>
  </kemet-accordion-panel>
</kemet-accordion>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-accordion.js';
import 'kemet-ui/dist/components/kemet-accordion-panel.js';
```

## API

<br />

### &lt;kemet-accordion&gt;
<ArgsTable of="kemet-accordion" />

#### Methods

| method         | description           |
| -------------- | --------------------- |
| .expandAll()   | Expands all panels.   |
| .collapseAll() | Collapses all panels. |

<br />

### &lt;kemet-accordion-panel&gt;
<ArgsTable of="kemet-accordion-panel" />

#### Methods

| method    | description                |
| --------- | -------------------------- |
| .toggle() | Opens or closes the panel. |

#### Slots

| slot    | description                                         |
| ------  | --------------------------------------------------- |
| trigger | The trigger button that opens and closes the panel. |
| icon    | The icon for the panel.                             |
| body    | The body content of the panel.                      |

#### CSS

| css                                      | description                                  | default |
| ---------------------------------------- | -------------------------------------------- | ------- |
| --kemet-accordion-panel-transition-speed | The speed of the opening and closing effect. | 0.3s    |
| ::part(trigger)                          | Contains the trigger for the panel.          |         |
| ::part(body)                             | Contains the body for the panel.             |         |
