# KemetScrollSnap

A scroll-snap implementation for responsive sliders.

## How to use

### Installation

```bash
npm install @kemet/kemet-scroll-snap
```

```js
import '@kemet/kemet-carousel/kemet-scroll-snap.js';
import '@kemet/kemet-carousel/kemet-scroll-snap-slide.js';
import '@kemet/kemet-carousel/kemet-scroll-snap-pagination.js'; // optional
```

### Basic Usage
```html preview-example
  <kemet-scroll-snap>
  <div slot="slides">
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
  </div>
</kemet-scroll-snap>
```

**Note:** if you're interested in hiding the scrollbar while in horizontal, like in our demo, here's how with css:

```css hide-scroll-bar
kemet-scroll-snap[axis="horizontal"] [slot="slides"] {
  scrollbar-width: none;
}

kemet-scroll-snap[axis="horizontal"] [slot="slides"]::-webkit-scrollbar {
  display: none;
  width: 0px;
  background: transparent;
}
```
