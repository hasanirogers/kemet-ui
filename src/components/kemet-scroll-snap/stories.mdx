# Kemet Scroll Snap

A scroll-snap implementation for responsive sliders.

## How to use
```html
<kemet-scroll-snap>
  <div slot="slides">
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
  </div>
  <kemet-scroll-snap-paginator slot="pagination"></kemet-scroll-snap-paginator>
</kemet-scroll-snap>
```

**Note:** If you're interested in hiding the scrollbar while in horizontal here's how to do it with css:

```css
kemet-scroll-snap[axis="horizontal"] [slot="slides"] {
  scrollbar-width: none;
}

kemet-scroll-snap[axis="horizontal"] [slot="slides"]::-webkit-scrollbar {
  display: none;
  width: 0px;
  background: transparent;
}
```
