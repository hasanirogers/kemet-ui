# Kemet Scroll Snap

A scroll-snap implementation for responsive sliders.

## How to use
```html
<kemet-scroll-snap>
  <div slot="slides">
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
    <kemet-scroll-snap-slide>[your content here]</kemet-scroll-snap-slide>
  </div>
  <kemet-scroll-snap-paginator slot="pagination"></kemet-scroll-snap-paginator>
</kemet-scroll-snap>
```

**Note:** If you're interested in hiding the scrollbar while in horizontal here's how to do it with css:

```css
kemet-scroll-snap[axis="horizontal"] [slot="slides"] {
  scrollbar-width: none;
}

kemet-scroll-snap[axis="horizontal"] [slot="slides"]::-webkit-scrollbar {
  display: none;
  width: 0px;
  background: transparent;
}
```

## API

<br />

### &lt;kemet-scroll-snap&gt;

<ArgsTable of="kemet-scroll-snap" />

#### Slots

| slot       | description                                         |
| ---------- | --------------------------------------------------- |
| slides     | Place your slides here.                             |
| pagination | Place the paginator component here if you want one. |

#### CSS

| css                                         | description                                | default |
| ------------------------------------------- | ------------------------------------------ | ------- |
| --kemet-scroll-snap-gap	                    | Space between the slides.                  | 1.5rem  |
| --kemet-scroll-snap-horizontal-max-width    | The horizontal max width of the container. | 1024px  |
| --kemet-scroll-snap-vertical-height         | The vertical height of the container.      | 100vh   |
| --kemet-scroll-snap-slides-vertical-padding | Padding on the vertical axis.              | 0 2rem  |
| ::part(slides)                              | The slides container.                      |         |
| ::part(pagination)                          | The pagination element.                    |         |

<br />

### &lt;kemet-scroll-snap-slide&gt;

<ArgsTable of="kemet-scroll-snap-slide" />

#### CSS

| css                             | description             | default |
| ------------------------------- | ----------------------- | ------- |
| --kemet-scroll-snap-slide-width | The width of the slide. | 100%    |
| --kemet-scroll-snap-slide-align | The slide alignment.    | center  |

<br />
