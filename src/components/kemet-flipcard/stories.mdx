# Kemet Flipcard

A card that has a front and back side which can be flipped.


## How to use
```html
<kemet-flipcard>
  <div slot="front">
    <p>This is the front of the card.</p>
  </div>
  <div slot="back">
    <p>This is the back of the card.</p>
  </div>
</kemet-flipcard>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-flipcard.js'
import 'kemet-ui/dist/components/kemet-flipcard-trigger.js'
```

#### CSS

| css                                     | description                                          | default                              |
| --------------------------------------- | ---------------------------------------------------- | ------------------------------------ |
| --kemet-flipcard-front-color	          | The text color for the front of the card.            | #202020                              |
| --kemet-flipcard-back-color	            | The text color for the back of the card.             | #202020                              |
| --kemet-flipcard-front-background-color	| The background color for the front of the card.      | #fafafa                              |
| --kemet-flipcard-back-background-color	| The background color for the back of the card.       | #fafafa                              |
| --kemet-flipcard-border                 | The border on the front and back of the card.        | 2px solid var(--kemet-color-primary) |
| --kemet-flipcard-border-radius          | The border radius on the front and back of the card. | 0                                    |
| --kemet-flipcard-ratio                  | The aspect raio of the card.                         | 16/9                                 |
| ::part(front)	                          | The front of the card.	                             |                                      |
| ::part(back)		                        | The back of the card.                                |                                      |
| ::part(wrapper)	                        | A container for both front and back of the card.	   |                                      |
