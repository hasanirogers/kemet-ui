# Kemet Flipcard

A card that has a front and back side which can be flipped.


## How to use
```html
<kemet-flipcard>
  <div slot="front">
    <p>This is the front of the card.</p>
  </div>
  <div slot="back">
    <p>This is the back of the card.</p>
  </div>
</kemet-flipcard>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-flipcard/kemet-flipcard.js'
import 'kemet-ui/dist/components/kemet-flipcard/kemet-flipcard-trigger.js'
```
