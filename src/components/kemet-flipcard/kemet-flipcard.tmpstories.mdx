# KemetFlipcard

A card that has a front and back side which can be flipped.


## How to use

### Installation

```bash
npm install @kemet/kemet-flipcard
```

```js
import '@kemet/kemet-flipcard/kemet-flipcard.js';
import '@kemet/kemet-flipcard/kemet-flipcard-trigger.js'; // optional
```

### Basic Usage
```html
<kemet-flipcard>
  <div slot="front">
    <p>This is the front of the card.</p>
  </div>
  <div slot="back">
    <p>This is the back of the card.</p>
  </div>
</kemet-flipcard>
```
