# Kemet Combo

Allows the user to select a choice filtered through an Input. May only be used as a component of Field.


## How to use

```html
<kemet-field>
  <kemet-input slot="input" name="input"></kemet-input>
  <kemet-combo slot="component" options='["Item 1", "Item 2", "Item 3"]'></kemet-combo>
</kemet-field>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-combo/kemet-combo.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-combo/kemet-combo.js" type="module"></script>
```

| ------------------------------------ | ------------------------------------- | ------------------------------------------------------------------- |
