# Kemet Select

An enhanced select element.


## How to use

```html
<kemet-select>
  <kemet-option label="Item 1" value="1"></kemet-option>
  <kemet-option label="Item 2" value="2"></kemet-option>
  <kemet-option label="Item 3" value="3"></kemet-option>
</kemet-select>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-select/kemet-select.js';
import 'kemet-ui/dist/components/kemet-select/kemet-option.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-select/kemet-selct.js" type="module"></script>
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-select/kemet-option.js" type="module"></script>
```
