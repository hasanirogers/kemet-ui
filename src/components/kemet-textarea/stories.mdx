# Kemet Textarea

An enhanced textarea element.


## How to use

```html
<kemet-textarea></kemet-textarea>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-textarea.js'
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-textarea.js" type="module"></script>
```

#### CSS

| css                                       | description                         | default                              |
| ----------------------------------------- | ----------------------------------- | ------------------------------------ |
| --kemet-textarea-padding                  | The padding on the textarea.        | 1rem                                 |
| --kemet-textarea-border                   | The border of the textarea.         | 1px solid var(--kemet-color-primary) |
| --kemet-textarea-border-color-error       | The border of the error state.      | 1px solid var(--kemet-color-error)   |
| --kemet-textarea-border-color-success     | The border of the success state.    | 1px solid var(--kemet-color-success) |
| --kemet-textarea-border-color-warning     | The border of the warning state.    | 1px solid var(--kemet-color-warning) |
| --kemet-textarea-border-radius-rounded    | The border radius on rounded.       | var(--kemet-border-radius)           |
| --kemet-textarea-border-filled            | The border on filled.               | none                                 |
| --kemet-textarea-color-filled             | The color on filled.                | var(--kemet-color-white)             |
| --kemet-textarea-background-color-filled  | The background-color on filled.     | var(--kemet-color-primary)           |
| --kemet-textarea-background-color-error   | The error state background color.   | var(--kemet-color-error)             |
| --kemet-textarea-background-color-success | The success state background color. | var(--kemet-color-success)           |
| --kemet-textarea-background-color-warning | The warning state background color. | var(--kemet-color-warning)           |
