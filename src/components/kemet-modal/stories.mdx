# Kemet Modal

A modal that has many built in effects and flexible styles.

## How to use

```html
<kemet-modal>
  <kemet-modal-close tabindex="0" role="button" aria-label="Close Button">
    <kemet-icon icon="x-circle-fill" size="24"></kemet-icon>
  </kemet-modal-close>
  <h2>Modal Title</h2>
  <p>Your modal contents here.<p>
</kemet-modal>
```

**Note:** `kemet-modal-close` is optional.

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-modal/kemet-modal.js';
import 'kemet-ui/dist/components/kemet-modal/kemet-modal-close.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-modal/kemet-modal.js" type="module"></script>
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-modal/kemet-modal-close.js" type="module"></script>
```

#### Methods

| method    | description                |
| --------- | -------------------------- |

#### CSS

| css                                     | description                           | default                          |
|---------------------------------------- |-------------------------------------- | -------------------------------- |
| --kemet-modal-content-min-width         | The minimum width of the content.     |  0                               |
| --kemet-modal-content-max-width         | The maximum width of the content.     |  none                            |
| --kemet-modal-content-background-color  | The background color of the content.  |  --kemet-color-white             |
| --kemet-modal-content-mobile-min-width  | The min with of the mobile content.   | 100%                             |
| --kemet-modal-content-mobile-min-height | The min height of the mobile content. | 80vh                             |
