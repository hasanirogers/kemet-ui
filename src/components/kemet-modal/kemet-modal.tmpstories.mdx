# KemetModal

**NOTE:** Your javascript is responsible for opening each `<kemet-modal>`. I recommend you open the modal by calling the `.open()` method rather than toggling the `opened` property. This way the `kemet-modal-open` event will fire when you open the modal.

### Installation

```bash
npm install @kemet/kemet-modal
```

```js
import '@kemet/kemet-modal/kemet-modal.js';
import '@kemet/kemet-modal/kemet-modal-close.js'; // optional if you want it
```


### Close when Modal is Clicked

Use the attribute `closeOnClick` to tell the modal to close when the over is clicked.

<Preview withToolbar>
  <Story name="Close when Modal Clicked">
    {html`
      <button @click=${() => openModal('closed-when-clicked')}>Open Modal</button>
      <kemet-modal id="closed-when-clicked" closeOnClick>
        <h1>Your Modal Contents</h1>
      </kemet-modal>
    `}
  </Story>
</Preview>

### Close with Kemet Modal Button

You can wrap your own content with `<kemet-modal-button>` and it'll act like a close button for the modal.

<Preview withToolbar>
  <Story name="Close with Kemet Close Button">
    {html`
      <button @click=${() => openModal('closed-with-button')}>Open Modal</button>
      <kemet-modal id="closed-with-button">
        <h1>Your Modal Contents</h1>
        <kemet-modal-close>
          <button>Close Modal</button>
        </kemet-modal-close>
      </kemet-modal>
    `}
  </Story>
</Preview>

### Modal with Effect Applied

Use the `effect` attrbitue along with a proper effect value to apply effects to the modal. Check under API for proper effect values.

<Preview withToolbar>
  <Story name="Effect Applied">
    {html`
      <button @click=${() => openModal('effect-applied')}>Open Modal</button>
      <kemet-modal id="effect-applied" closeOnClick effect="super-scaled">
        <h1>This modal has a super-scaled effect applied.</h1>
      </kemet-modal>
    `}
  </Story>
</Preview>
