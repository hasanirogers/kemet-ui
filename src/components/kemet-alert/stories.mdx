# Kemet Alert

Calls out important messages and notifications.

## How to use

```html
<kemet-alert opened>
  The brown fox jumped over the lazy dog.
</kemet-alert>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-alert/kemet-alert.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-alert/kemet-alert.js" type="module"></script>
```
