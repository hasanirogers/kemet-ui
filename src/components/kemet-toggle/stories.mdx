# Kemet Toggle

A toggle swtich for forms.


## How to use

```html
<kemet-toggle></kemet-toggle>
```

## How to import

<br />

### JavaScript
```javascript
import 'kemet-ui/dist/components/kemet-toggle/kemet-toggle.js';
```

### HTML
```html
<script src="https://unpkg.com/kemet-ui@latest/dist/components/kemet-toggle/kemet-toggle.js" type="module"></script>
```
