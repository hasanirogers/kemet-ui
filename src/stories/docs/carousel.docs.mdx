# Kemet Carousel

The Carousel allows you to click or swipe through content.

<kemet-tabs panel-effect="stacked" divider>
  <kemet-tab slot="tab">Usage</kemet-tab>

    ### Markup
    ```html
    <kemet-carousel>
      <kemet-carousel-slide slot="slides">Slide 1</kemet-carousel-slide>
      <kemet-carousel-slide slot="slides">Slide 2</kemet-carousel-slide>
      <kemet-carousel-slide slot="slides">Slide 3</kemet-carousel-slide>
    </kemet-carousel>
    ```

### Options
The options object contains a number of settings for the carousel. Use these options to set the default options for the carousel. You can override them at specified breakpoints in the breakpoints object.
<table>
  <thead>
    <tr>
      <th>Option</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>perView</td>
      <td>The number of slides to display in the container. Access decimals numbers.</td>
    </tr>
    <tr>
      <td>perMove</td>
      <td>The number of slides to move on next and prev shifts.</td>
    </tr>
    <tr>
      <td>gap</td>
      <td>The number, in pixels, of space between each slide.</td>
    </tr>
    <tr>
      <td>slideshow</td>
      <td>The number, in seconds, to display a slide before rotating to a new one. Use 0 to disable a slideshow.</td>
    </tr>
    <tr>
      <td>rewind</td>
      <td>Loops back to the first or last slide when a limit is reached.</td>
    </tr>
    <tr>
      <td>center</td>
      <td>Centers the active slide.</td>
    </tr>
  </tbody>
</table>
